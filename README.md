# Features
- Take pictures using webcam
- Choose from different filters
- Pick the color of the photostrip
- Download the photostrip as a PNG file

# Tech Stack
- Next.js
- Javascript
- TailwindCSS
- Framer Motion
- downloadjs
- html2canvas
